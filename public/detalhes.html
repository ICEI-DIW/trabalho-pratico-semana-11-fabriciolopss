Sobre Projeto desenvolvido para a disciplina de Desenvolvimento de interfaces web, com o intuito de ser uma plataforma de eventos, onde o usuário pode criar e visualizar eventos. O projeto foi desenvolvido utilizando HTML, CSS e JavaScript puro, sem o uso de frameworks ou bibliotecas externas. O objetivo é proporcionar uma experiência simples e intuitiva para o usuário, permitindo que ele encontre eventos de forma rápida e fácil.
